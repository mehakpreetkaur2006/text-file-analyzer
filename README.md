# Text File Analyzer Website

## Description
A web-based Text File Analyzer developed using Python (Flask), HTML, CSS, and JavaScript.  
It analyzes text and provides statistics like word count, sentence count, and most frequent words.

## Features
- Word count
- Character count
- Line count
- Sentence count
- Top repeated words

## Technologies Used
- Python (Flask)
- HTML
- CSS
- JavaScript
